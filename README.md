# SimplePython
Work for compilers discipline, With the use of antlr to build a simplified python language.

## Tools
* Pycharm
* Antlr4
* Python3

## Implemented Structures
- [x] control command
- [x] Types and Operators
- [x] Identifiers
- [x] Functions
- [x] Main Program
- [X] Variables and Constants
